# Clínica Fiorentini - Assets Directory

## Hero Image

The hero image should feature:

- **Subjects**: Professional female dentist and male physiotherapist
- **Expression**: Warm, friendly smiles conveying trust and approachability
- **Setting**: Modern medical clinic with teal/seafoam green accents
- **Lighting**: Natural, bright, professional
- **Dimensions**: Minimum 1200x800px (landscape orientation)
- **Format**: WebP preferred, JPG fallback
- **Style**: High-end commercial photography, 8k quality

## Image Placement

Replace the placeholder in `templates/index.html`:

```html
<img src="../assets/hero-image.png" alt="Equipe Clínica Fiorentini">
```

## Recommended Image Sources

1. **Professional Photography**: Hire a local photographer for authentic team photos
2. **Stock Photography**: 
   - Unsplash (free, high-quality)
   - Pexels (free)
   - Adobe Stock (premium)
   - Shutterstock (premium)

## Search Keywords for Stock Photos

- "medical team smiling"
- "dentist and physiotherapist"
- "healthcare professionals portrait"
- "clinic team modern"
- "medical staff friendly"

## Image Optimization

Before using images:

1. **Resize**: Scale to appropriate dimensions (1200x800 for hero)
2. **Compress**: Use tools like TinyPNG or Squoosh
3. **Format**: Convert to WebP with JPG fallback
4. **Alt Text**: Always include descriptive alt text for accessibility

## Team Photos

For the team section, you'll need:

- **Dimensions**: 300x300px (square)
- **Format**: WebP or JPG
- **Style**: Professional headshots with clean background
- **Consistency**: Same lighting and background style for all team members

## Color Grading Tips

To match the clinic's aesthetic:

- Enhance teal/green tones slightly
- Maintain natural skin tones
- Increase brightness for clean, sterile feel
- Reduce warm tones slightly for modern look

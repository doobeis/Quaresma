## Typography

- **Headings**: Poppins (700 weight)
- **Body**: Inter (300-700 weights)

## Key Features Demonstrated

1. **Hero Section**: Professional team imagery with gradient overlay
2. **Services Cards**: Hover effects with seafoam green accents
3. **Team Section**: Glassmorphic cards with professional photos
4. **Appointment Form**: Clean, accessible form design
5. **WhatsApp Integration**: Floating action button
6. **Responsive Design**: Mobile-first approach

## Usage

To view the landing page:

1. Open `templates/index.html` in a modern web browser
2. Replace placeholder images with actual professional photography
3. Update contact information and team details
4. Customize service offerings as needed

## Image Requirements

For best results, use:
- **Hero Image**: 1200x800px minimum, professional team photo
- **Team Photos**: 300x300px minimum, square format, professional headshots
- **Format**: WebP or optimized JPG
- **Style**: Clean, professional, well-lit medical environment

## Customization Tips

1. **Colors**: All colors use CSS custom properties for easy theming
2. **Content**: Update text directly in HTML
3. **Services**: Add/remove service cards in the services section
4. **Team**: Add more team members by duplicating team-card divs
5. **Form**: Integrate with your backend or email service

## Performance Optimization

- Use WebP images with JPG fallbacks
- Implement lazy loading for images
- Minify CSS and JavaScript for production
- Enable gzip compression on server
- Use CDN for font delivery

## Accessibility

The template includes:
- Semantic HTML5 elements
- ARIA labels where needed
- Keyboard navigation support
- Sufficient color contrast ratios
- Responsive font sizing

# Clínica Fiorentini Landing Page Skill

## Overview
This skill provides a complete, high-end landing page design for **Clínica Fiorentini**, a dual-specialty clinic offering both Odontology and Physiotherapy services. The design emphasizes trust, professionalism, and modern healthcare aesthetics.

## Design Specifications

### Color Palette
- **Primary**: Teal (#0D9488, #14B8A6)
- **Secondary**: Seafoam Green (#5EEAD4, #99F6E4)
- **Neutral**: Soft Grey (#F3F4F6, #E5E7EB)
- **Accent**: White (#FFFFFF)
- **Text**: Charcoal (#1F2937, #374151)

### Visual Style
- **Layout**: Modern minimalist, clean white background with teal accents
- **Typography**: Clean, professional sans-serif (Inter, Poppins)
- **Photography**: Professional, high-quality imagery conveying trust
- **Aesthetic**: Sleek, sterile, medical-grade professionalism
- **Resolution**: 8k-ready, optimized for all devices

### Key Sections

#### 1. Hero Section
- Professional female dentist and male physiotherapist
- Smiling, approachable, conveying trust
- Clean headline with clinic name
- Primary CTA button
- Subtle teal gradient overlay

#### 2. Services Section
- Dual specialty showcase (Odontology & Physiotherapy)
- Icon-based service cards
- Hover effects with seafoam green accents
- Brief descriptions of key treatments

#### 3. Our Team Section
- Professional team photos
- Credentials and specializations
- Trust-building bios
- Glassmorphic card design

#### 4. Book an Appointment Section
- Prominent contact form
- WhatsApp integration
- Calendar visualization
- Clear CTA with teal button

## Usage

### Quick Start
1. Copy the HTML template from `templates/index.html`
2. Customize the content with actual clinic information
3. Replace placeholder images with professional photography
4. Deploy to web hosting

### Customization Points
- **Clinic Name**: Update in hero section and navbar
- **Team Members**: Add actual staff photos and credentials
- **Services**: Customize treatment offerings
- **Contact Information**: Update phone, email, WhatsApp
- **Location**: Add Google Maps integration

## Technical Stack
- **HTML5**: Semantic markup
- **CSS3**: Custom properties, Grid, Flexbox

- **Responsive**: Mobile-first design
- **Performance**: Optimized images, lazy loading

## Best Practices
1. Use high-quality, professional photography
2. Maintain consistent teal/seafoam color scheme
3. Ensure WCAG 2.1 AA accessibility compliance
4. Optimize images for web (WebP format recommended)
5. Test on multiple devices and browsers
6. Implement proper meta tags for SEO
